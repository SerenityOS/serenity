## Name

GML Horizontal Box Layout

## Description

Defines a horizontal GUI box layout. This is a layout object that lays out widgets side-by-side horizontally.

## Examples

```gml
@GUI::Widget {
    layout: @GUI::HorizontalBoxLayout {
        spacing: 2
    }
}
```
