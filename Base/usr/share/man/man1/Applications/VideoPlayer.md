## Name

![Icon](/res/icons/16x16/app-video-player.png) VideoPlayer

[Open](file:///bin/VideoPlayer)

## Synopsis

```**sh
$ VideoPlayer [file]
```

## Description

`Video Player` is an application for playing videos.

## Arguments

* `file`: The video file to be played.

## Examples

```sh
$ VideoPlayer /res/videos/scene.wav
```
